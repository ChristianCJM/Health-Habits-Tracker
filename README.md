# BalanceUp

BalanceUp is a mobile health habit tracker developed as part of an academic project.

## Project Structure

- `frontend/` – Flutter mobile application
- `backend/` – Java REST API
- Database – Supabase (PostgreSQL)

## Main Features

- User authentication (login & register)
- Habit creation and tracking
- Daily progress logging
- Statistics visualization
- Local device notifications (12:00 and 18:00 reminders)

## Technologies

- Flutter / Dart
- Java
- Supabase (PostgreSQL)
- REST API
- Local Notifications

## How to Run

### Backend
- Configure database connection
- Run Java application

### Frontend
- `flutter pub get`
- `flutter run`
